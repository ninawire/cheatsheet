Examples


Here we have some examples of how to use CSS selectors in your projects.


Simple Selectors

CSS element Selector

Here, all <p> elements on the page will be center-aligned, with a red text color:

p {

text-align: center;

color: red;

}

CSS id Selector

The CSS rule below will be applied to the HTML element with id="para1":

#para1 {

text-align: center;

color: red;

}

CSS class Selector

In this example all HTML elements with class="center" will be red and center-aligned:

.center {

text-align: center;

color: red;

}

You can also specify that only specific HTML elements should be affected by a class. In this example only <p> elements with class="center" will be red and center-aligned:

p.center {

text-align: center;

color: red;

}

HTML elements can also refer to more than one class.In this example the <p> element will be styled according to class="center" and to class="large":

<p class="center large">This paragraph refers to two classes.</p>

CSS Universal Selector

The CSS rule below will affect every HTML element on the page:

* {

text-align: center;

color: red;

}

CSS Grouping Selector

In this example we have grouped the selectors to minimize the code:

h1, h2, p {

text-align: center;

color: red;

}

CSS Combinators Selector

The descendant selector matches all elements that are descendants of a specified element:

div p {

text-align: center;

color: red;

}

The child selector selects all elements that are the children of a specified element:

div > p {

text-align: center;

color: red;

}

The adjacent sibling selector is used to select an element that is directly after another specific element:

div + p {

text-align: center;

color: red;

}

The general sibling selector selects all elements that are next siblings of a specified element.:

div ~ p {

text-align: center;

color: red;

}

CSS Pseudo-classes Selector

The syntax of pseudo-classes:

selector:pseudo-class {

property: value;

}

CSS Pseudo-elements Selector

The syntax of pseudo-elements:

selector::pseudo-element {

property: value;

}

CSS [attribute] Selector

The syntax of pseudo-elements:

a[target] {

background-color: yellow;

}
